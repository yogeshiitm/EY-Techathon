Welcome to TechHD Covid19 website!

What is TechHD for?

Here we have tried to help the government make informed decisions around epidemiological and vaccine supply circumstances by predicting the more important segments of our country that need to be catered with vaccine deliveries as prior as possible.

How it works?

  • We went through various census report estimates, WHO reports, and other state websites to fetch and combine all these data features and made our datasets.
  • We then add and update the data daily with the number of covid cases in each state and carry out our ML algorithms.
  • And hence we ended up with the following features that we used for batch predictions:
    1. Active Cases (covid19india.org website APIs)
    2. Population density (estimated: since the census is done every 10 years(2021 next))
    3. Death rate (no.of deaths/total confirmed cases x100)
    4. Heath Workers Present in the State (WHO reports)
    5. Senior Citizens(60+)
    6. Children(0-14yrs)
    7. Allotted Hospital Beds for Covid Patients in each state
    8. Accessibility of each state( based on National Highways lengths and connections within the state
  • We then run our ML clustering algorithms through all the features and load them into our ranking algorithms (considering several metrics and features) to figure out the Importance and urgency of vaccine deliveries in each state.
  • Our datasets are automatically updated daily to accommodate and make the predictions based on the latest information regarding the number of active covid cases of each state.

What can you find on this website?
HomePage:
  • The Table shows information about various states and their corresponding Batch Number, i.e., the state’s priority level.
  • For example, if Maharashtra and Kerala are in batch no. 1, then the first lot of vaccine deliveries should be delivered to those states. And then the remaining states in Batch number 2 and so on.
District Page:
  • This Page provides us further segmentation of each state into various batches(priority sections) as predicted by our powerful ML algorithms.
  • This would provide further assistance to the State Government as well in delivering the Vaccines with the state.
Batch Page:
  • This Page gives the batch-wise analysis and indicates the further percentage distribution of vaccines to each state in a particular batch.
  • For a particular Batch, the percentage column in the table indicates the percentage of the “vaccine-supplies” distributed among each state in that batch.
  • For example, In the batch 1 table, we see Maharashtra and Kerala and their 30% and 70% percentages. This predicts that while delivering the first lot, 70% of the total vaccine supplies to be delivered to Kerala and the remaining 30% of that lot to Maharashtra.
  • The Percentage Values in each batch have been predicted based on their priority value given by our ranking algorithm.
  • We formulated the ranking Algorithm by considering various factors(shown above) and then using the proper weighing system for each aspect.
Note to the reader:

Our ML and AI algorithms’ predictions are made by considering various factors, such as the estimated population datasets, India Census surveys and trends, the Number of Active Cases each day from trusted websites and APIs, WHO reports on the number of health workers in the state, Number of Hospital beds designated to Covid patients, Children(0-14) and Senior Citizens(60+) numbers, the Accessibility of states using National Highways lengths and connections in the state, etc.
